# Jakub Rimmer | Portfolio

A simple personal portfolio website showcasing selected software engineering projects with a focus on backend systems, security, reliability, and performance.

## Projects featured
- **FinWiz** (academic, team of 4)  
  Backend + deployment work, CI/CD, secure handling of inputs, databases.
- **Distributed Search Engine** (academic, team of 4)  
  Crawler + AWS integration, integration testing, logging/observability, code reviews.
- **PennOS** (academic)  
  Educational OS in C focused on processes, scheduling, and file operations.
- **RISC-V CPU and Cache** (academic, team of 2)  
  SystemVerilog CPU and cache oriented work with verification via automated tests.
- **Parallel Algorithmic Patterns in Java (Dissertation)**  
  Algorithmic skeletons, benchmarking with JMH, unit tests, concurrency comparisons.
- **Hackathons (PennApps and others)**  
  Authentication in prototypes and security-minded design under tight timelines.

Note: Some academic project code and reports are private due to course policy. High-level details are available on request.
